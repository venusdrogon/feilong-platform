1359     /**
1360      * 判断某年是否为闰年 .
1361      * 
1362      * <p>
1363      * 规则: {@code (year % 4 == 0 && year % 100 != 0) || year % 400 == 0}
1364      * </p>
1365      * 
1366      * <h3>闰年原因:</h3>
1367      * 
1368      * <blockquote>
1369      * 
1370      * <p>
1371      * 地球绕太阳运行周期为<b>365天5小时48分46秒</b>(合365.24219天)即一回归年(tropical year).<br>
1372      * 公历的平年只有365日,比回归年短约0.2422日,所余下的时间约为四年累计一天,故四年于2月加1天,使当年的历年长度为366日,这一年就为闰年.<br>
1373      * </p>
1374      * 
1375      * <p>
1376      * 现行公历中每400年有97个闰年.按照每四年一个闰年计算,平均每年就要多算出0.0078天,这样经过四百年就会多算出大约3天来,<br>
1377      * 因此,每四百年中要减少三个闰年.
1378      * </p>
1379      * 
1380      * <p>
1381      * 所以规定,公历年份是整百数的,必须是400的倍数的才是闰年,不是400的倍数的,虽然是100的倍数,也是平年,<br>
1382      * 这就是通常所说的:<b>四年一闰,百年不闰,四百年再闰</b>.<br>
1383      * 
1384      * 例如,2000年是闰年,1900年则是平年.<br>
1385      * </p>
1386      * </blockquote>
1387      * 
1388      * @param year
1389      *            年份
1390      * @return 四年一闰,百年不闰,四百年再闰
1391      * @see GregorianCalendar#isLeapYear(int)
1392      */
1393     public static boolean isLeapYear(int year){
1394         return new GregorianCalendar().isLeapYear(year);
1395     }
