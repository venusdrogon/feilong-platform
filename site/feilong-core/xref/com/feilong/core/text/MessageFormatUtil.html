42      /**
43       * 调用 {@link java.text.MessageFormat#format(String, Object...)} 格式化.
44       * 
45       * <h3>用法:</h3>
46       * 
47       * <blockquote>
48       * 
49       * <pre class="code">
50       * MessageFormatUtil.format("name=张三{0}a{1}", "jin", "xin")                                             =   "name=张三jinaxin"
51       * MessageFormatUtil.format("name=张三{0,number}a{1}", 5, "xin")                                          =   "name=张三5axin"
52       * MessageFormatUtil.format("name=张三{0,date,yyyy-MM-dd}a{1}", DateUtil.toDate("2000", DatePattern.yyyy), "xin")    =   "name=张三2000-01-01axin"
53       * </pre>
54       * 
55       * </blockquote>
56       * 
57       * <h3>关于 参数pattern(模式参数):</h3>
58       * 
59       * <blockquote>
60       * <p>
61       * 格式： ArgumentIndex[,FormatType[,FormatStyle]]
62       * </p>
63       * 
64       * <p>
65       * 占位符有三种方式书写方式:
66       * </p>
67       * 
68       * <ul>
69       * <li>{argumentIndex}: 0-9之间的数字,表示要格式化对象数据在参数数组中的索引号</li>
70       * <li>{argumentIndex,formatType}:参数的格式化类型</li>
71       * <li>{argumentIndex,formatType,FormatStyle}: 格式化的样式,它的值必须是与格式化类型相匹配的合法模式、或表示合法模式的字符串.</li>
72       * </ul>
73       * 
74       * <table border="1" cellspacing="0" cellpadding="4" summary="">
75       * 
76       * <tr style="background-color:#ccccff">
77       * <th align="left">字段</th>
78       * <th align="left">说明</th>
79       * </tr>
80       * 
81       * <tr valign="top">
82       * <td>ArgumentIndex</td>
83       * <td>是从0开始的入参位置索引</td>
84       * </tr>
85       * 
86       * <tr valign="top" style="background-color:#eeeeff">
87       * <td>FormatType</td>
88       * <td>指定使用不同的Format子类对入参进行格式化处理。值范围如下：
89       * 
90       * <table border="1" cellspacing="0" cellpadding="4" summary="">
91       * <tr style="background-color:#ccccff">
92       * <th align="left">字段</th>
93       * <th align="left">说明</th>
94       * </tr>
95       * 
96       * <tr valign="top">
97       * <td>number</td>
98       * <td>调用{@link java.text.NumberFormat}进行格式化</td>
99       * </tr>
100      * 
101      * <tr valign="top" style="background-color:#eeeeff">
102      * <td>date</td>
103      * <td>调用{@link java.text.DateFormat}进行格式化</td>
104      * </tr>
105      * <tr valign="top">
106      * <td>time</td>
107      * <td>调用{@link java.text.DateFormat}进行格式化</td>
108      * </tr>
109      * 
110      * <tr valign="top" style="background-color:#eeeeff">
111      * <td>choice</td>
112      * <td>调用{@link java.text.ChoiceFormat}进行格式化</td>
113      * </tr>
114      * 
115      * </table>
116      * 
117      * </td>
118      * 
119      * </tr>
120      * <tr valign="top">
121      * <td>FormatStyle</td>
122      * <td>设置FormatType中使用的格式化样式。<br>
123      * 值范围如下：
124      * short,medium,long,full,integer,currency,percent,SubformPattern(子格式模式,形如#.##)</td>
125      * </tr>
126      * </table>
127      * 
128      * <p>
129      * 注意： FormatType 和 FormatStyle 主要用于对日期时间、数字、百分比等进行格式化。
130      * </p>
131      * 
132      * <pre>
133      * 示例——将数字1.23格式为1.2：
134      * 
135      * double num = 1.23;
136      * String str = MessageFormatUtil.format("{0,number,#.#}", num);
137      * </pre>
138      * 
139      * </blockquote>
140      * 
141      * 
142      * <h3>注意点:</h3>
143      * 
144      * <blockquote>
145      * <ol>
146      * <li>两个单引号才表示一个单引号,仅写一个单引号将被忽略。</li>
147      * <li>单引号会使其后面的占位符均失效,导致直接输出占位符。
148      * 
149      * <pre>
150      * MessageFormatUtil.format("{0}{1}", 1, 2); // 结果12
151      * MessageFormatUtil.format("'{0}{1}", 1, 2); // 结果{0}{1}
152      * MessageFormatUtil.format("'{0}'{1}", 1, 2); // 结果{0}
153      * 因此可以用于输出左花括号(单写左花括号会报错,而单写右花括号将正常输出)
154      * 
155      * MessageFormatUtil.format("'{'{0}}", 2); // 结果{2
156      * </pre>
157      * 
158      * </li>
159      * </ol>
160      * </blockquote>
161      * 
162      * @param pattern
163      *            模式参数
164      * @param arguments
165      *            动态参数
166      * @return 如果 <code>pattern</code> 是null,抛出 {@link NullPointerException}<br>
167      */
168     public static String format(String pattern,Object...arguments){
169         Validate.notNull(pattern, "pattern can't be null!");
170         return MessageFormat.format(pattern, arguments);
171     }
